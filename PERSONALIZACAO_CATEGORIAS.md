# Personalização por Categoria — Estratégia de Design

## Conceito

Cada categoria profissional tem sua própria **personalidade visual, templates, e funcionalidades específicas** que agregam valor ao app.

Quando o usuário seleciona uma categoria, o app inteiro se adapta:
- Cores de tema
- Ícones e elementos visuais
- Dashboard layout
- Templates de dados
- Ferramentas específicas

## IMPLEMENTAÇÃO TÉCNICA

### 1. Estrutura de Dados

```javascript
const categoryConfig = {
  'sound-technician': {
    id: 'sound-technician',
    label: 'Técnico de Som',
    icon: 'Zap', // Lucide icon
    color: '#39FF14', // Neon Green
    accentColor: '#00D9FF',
    backgroundImage: 'https://unsplash.com/...', // Foto técnico de som
    templates: [
      { id: 'basic', name: 'Setup Básico', items: [...] },
      { id: 'pro', name: 'Setup Profissional', items: [...] }
    ],
    dashboardKPIs: ['successRate', 'equipmentUsage', 'setupTime'],
    tools: ['soundCheck', 'levelMeter', 'equipmentChecklist'],
    customFields: [
      { name: 'dBLevel', type: 'number' },
      { name: 'setupNotes', type: 'text' }
    ]
  },
  'lighting': {
    id: 'lighting',
    label: 'Iluminação',
    icon: 'Lightbulb',
    color: '#A64AFF',
    // ...
  }
  // ... outras categorias
}
```

### 2. Layout Dinâmico por Categoria

```jsx
function Dashboard() {
  const category = useCategory(); // Pega categoria do usuário
  const config = categoryConfig[category.id];

  return (
    <div style={{ 
      --primary-color: config.color,
      --accent-color: config.accentColor 
    }}>
      <DashboardLayout>
        {config.dashboardKPIs.map(kpi => (
          <KPICard key={kpi} kpi={kpi} category={category} />
        ))}
      </DashboardLayout>
    </div>
  );
}
```

## AGREGAÇÃO DE VALOR

### 1. Ferramentas Específicas por Categoria
- **Som**: Calculadora de dB, guia de SPL, setlist
- **Luz**: Paleta RGB, temperatura de cor, spread calculator
- **Foto**: F-stops, shutter speed, ISO guide
- **Vídeo**: Bitrate calculator, color grading tools
- **DJ**: BPM converter, key finder, harmonic mixing
- **Produtor**: Budget tracker, supplier database
- **Design**: Color palette generator, material cost calc

### 2. Templates Inteligentes
Cada template vem com:
- Checklist pré-preenchido
- Valores típicos (time estimates, equipment list)
- Notas baseadas em experiência
- Links para suppliers/rentals

### 3. Comunidade & Sharing
- Compartilhar templates com colegas
- Ver setups de outros profissionais
- Rating de templates (qual funciona melhor)
- Best practices por categoria

### 4. Integração com Fornecedores
- Locar equipamento direto do app
- Integração com suppliers da região
- Histórico de rentals e preços
- Recomendações baseadas em perfil

## RESUMO

O Backstage Pro não é só um app genérico de gestão de eventos — é **especializado em cada profissão**, com ferramentas, templates, e visual adaptado.

Isso agrega valor porque:
✅ Usuário não começa do zero  
✅ App se adapta a cada profissão  
✅ Features específicas economizam tempo  
✅ Comunidade compartilha conhecimento  
✅ Integração com suppliers facilita workflow  

**Resultado**: App não é "mais um CRM" — é um **companheiro inteligente para cada profissão**.
